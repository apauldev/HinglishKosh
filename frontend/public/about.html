About HinglishKosh

What is Hinglish?

Hinglish is the code-mixed blend of Hindi and English widely spoken and written across India. It uses the Roman script without diacritics — "chai" instead of "cāy", "namaste" instead of "namastē". Over 600 million people communicate this way daily, making it one of the most widely used language varieties in the world.

The Problem

Despite its scale, Hinglish has no standardized dictionary. Academic resources use ISO 15919 transliteration with diacritics, which doesn't match how people actually type on phones. Commercial keyboards keep their dictionaries proprietary. Open-source keyboards lack adequate Hindi/Hinglish support.

The Solution

HinglishKosh is a 209,462-entry Hinglish-English dictionary that maps Devanagari headwords to their informal romanized spellings. It's built from two authoritative sources:

  • Hindi WordNet (IIT Bombay) — 153,204 core entries with synset linkages
  • Wiktionary (kaikki.org) — 56,258 extended entries

The dataset is processed through a pipeline that normalizes, transliterates, merges, deduplicates, and safety-filters every entry.

Technical Stack

This website is built with:

  • Python — data pipeline (ingestion, transliteration, merge, safety filter)
  • SQLite / D1 — full-text search database with FTS5
  • Cloudflare Pages + D1 — serverless hosting with edge functions

The full source code is available on GitHub.

Usage

Use the search bar above to look up any word. You can search by Devanagari (पानी), Hinglish romanization (paani), or English definition (water). Each word page shows the definition, part of speech, example sentences, and related words.
